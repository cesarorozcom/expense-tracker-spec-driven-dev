CSV File Format

Your CSV file should have the following columns:

  • account_name - Account name (required unless default selected below)
  • date - Date in YYYY-MM-DD or YYYY-MM-DD HH:MM format (required)
  • amount - Transaction amount (required)
  • description - Transaction description (optional)

Example:

account_name,date,amount,description
Checking,2026-05-16 10:00,25.50,Coffee
Checking,2026-05-16,15.00,Lunch
Savings,2026-05-15,100.00,Transfer
